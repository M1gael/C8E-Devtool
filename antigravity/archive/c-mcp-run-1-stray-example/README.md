# Tina4 Python Example

This example demonstrates how to use the Tina4 Python framework for routing, ORM models, and database querying based on the context retrieved from the `tina4_context` tool.

## Key Concepts Illustrated

1. **Routing**: Defining public and secured HTTP routes using `@get` and `@post`.
2. **ORM Models**: Creating database models using the Active Record ORM pattern.
3. **Database Queries**: Querying the database with bindings.

### File structure:
- [app.py](file:///c:/Users/work/Documents/projects/C8E-Devtool/example/app.py): The main entry point initializing the database and starting the server.
- [models.py](file:///c:/Users/work/Documents/projects/C8E-Devtool/example/models.py): Defining the ORM models.
- [routes.py](file:///c:/Users/work/Documents/projects/C8E-Devtool/example/routes.py): Defining the routing handlers.
